# Agent Browser Skill

This skill allows Claude Code to automate browser interactions using the `agent-browser` tool.

## Installation

This skill is automatically installed via the `setup.sh` script in the root of this repository.

To install it manually:

1. Copy this folder to `~/.claude/skills/agent-browser`
2. Ensure the `agent-browser` tool is installed in your system path.

## Usage

Once installed, Claude will be able to use the `agent-browser` command to interact with websites. See `SKILL.md` for full command documentation.
