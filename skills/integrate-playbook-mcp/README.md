# Integrate Playbook MCP Skill

This skill allows AI agents to expertly integrate the **Agents Playbook** Model Context Protocol (MCP) server into various AI frameworks and design production-ready workflows.

## Setup

1. Ensure you have the `agents-playbook` MCP server running or accessible.
2. Install the necessary adapters for your target framework:
   - **Vercel AI SDK**: `npm install @ai-sdk/mcp`
   - **LangChain**: `npm install @langchain/mcp-adapters`
   - **OpenAI**: Native support (see documentation)

## Usage

Assign the **Playbooker** agent to tasks involving:
- Setting up new MCP connections.
- Designing multi-stage AI pipelines.
- Updating workflows for 2026 models.
- Implementing context-preservation strategies like Memory Boards.
